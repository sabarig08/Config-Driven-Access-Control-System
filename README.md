# Config-Driven-Access-Control-System

# RBAC Project

## Overview
Config-driven Role Based Access Control system with table-level and field-level permissions.

## Tech Stack
- Frontend: React
- Backend: Node.js
- Database: MySQL

## Features
- Admin role management
- Role-based UI rendering
- Backend permission enforcement

## Steps to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
